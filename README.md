## ✨ What this tool does

humanise-text-skill focuses on three things:

- It checks for banned words that often make writing feel machine-made.
- It looks for repeated structure that makes text sound flat.
- It runs a 12-check validation pass to catch common writing patterns.

This makes it useful for blog drafts, notes, product copy, emails, and other text that needs a more natural voice.

## 🛠️ How It Works

The skill looks at your text in layers.

1. It scans for banned terms.
2. It checks sentence shape.
3. It looks for repeated habits in the writing.
4. It runs a final validation step.
5. It returns text that feels less robotic and more direct.

This gives you a simple way to improve a draft without starting over.

## 📋 Basic Workflow

1. Start with your draft.
2. Run it through the skill.
3. Review the changes.
4. Keep any parts that still sound like you.
5. Paste the result into your final document.

This works best when you already have a rough draft and want to make it sound more natural.

## 🔍 Validation Checks

The 12-check validation step helps catch issues such as:

- too many repeated sentence starts
- too many abstract terms
- too much passive voice
- overuse of filler phrases
- weak or vague phrasing
- phrases that sound copied from AI output
- list-heavy structure
- forced transitions
- empty claims
- flat rhythm
- long, dense blocks of text
- wording that feels too polished and generic
